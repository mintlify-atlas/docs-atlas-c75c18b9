# Welcome to Glass

Glass is a fast, modern code editor built on the foundation of [Zed](https://zed.dev), designed for developers who value speed, simplicity, and powerful AI integration.

<Note>
  Glass is a fork of Zed Industries' Zed editor, enhanced with additional features and optimizations for development workflows.
</Note>

## Why Glass?

Glass combines the best aspects of modern code editors with cutting-edge AI capabilities:

<CardGroup cols={2}>
  <Card title="Blazing Fast Performance" icon="bolt">
    Written in Rust and built on the GPUI framework for native performance. Opens instantly, handles large codebases effortlessly.
  </Card>
  
  <Card title="AI-Powered Development" icon="brain">

  </Card>

## Key Features

### Intelligent Editing

- **Tree-sitter powered** syntax highlighting and code navigation
- **LSP integration** for diagnostics, completions, and refactoring
- **Multi-cursor editing** with smart selection expansion
- **Project-wide search** and replace with regex support

### AI Integration

- **Agent Panel**: Conversational AI assistant with tool access
- **Inline Assistant**: Context-aware code generation and editing
- **Edit Prediction**: Real-time AI suggestions as you type
- **Multiple Providers**: Support for Anthropic, OpenAI, Ollama, and more

### Developer Tools

- **Integrated Terminal** with multiple shell support
- **Git Integration** with inline diff views and commit workflows
- **Debugger** support with breakpoints and variable inspection
- **Task Runner** for build scripts and custom commands

### Collaboration

- **Real-time editing** with multiple users
- **Voice chat** and screen sharing
- **Shared projects** with synchronized state

## Supported Platforms

Glass runs on modern operating systems with native performance:

- **macOS** 12 (Monterey) and later
- **Linux** with Vulkan 1.3 support (x86_64 and aarch64)
- **Windows** 10 (version 1903) and later
